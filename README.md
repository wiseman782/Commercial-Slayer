# CommercialSlayer

I don't like commercials. I like music.

CommercialSlayer is a Chrome extension that:

- automatically mutes Peacock during ad breaks and switches your audio to Spotify
- automatically restores Peacock audio and pauses Spotify when the commercial break ends

## How It Works

1. Open Peacock in a browser tab and start watching.
2. Open Spotify Web Player in another tab.
3. Start playback in Spotify (Chrome requires this once to allow autoplay).
4. Open the extension popup.
5. Assign the Peacock tab and the Spotify tab.
6. Click Turn On.

That's it.

## Requirements

- Chrome or Chromium-based browser
- Peacock web player
- Spotify Web Player
